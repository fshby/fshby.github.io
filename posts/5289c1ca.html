python的pytest解析器

pip list —查看终端解释器

Set-ExecutionPolicy RemoteSigned —设置终端，可以打开终端

pytest框架编码规范：

类以Test开头
测试用例必须是test开头

运行方式：
①pycharm界面运行
终端命令：
②pytest -s -v 文件路径
案例：pytest -s -v .\pytest框架\demo.py
③if name == ‘main‘:
pytest.main([‘-s’,’-v’,’demo.py’])

创建测试用例目录 testCase
在根目录创建配置文件 pytest.ini
配置pytest.ini文件
注意：配置文件中addopts 命令行不能有空格
[pytest]
addopts = -s -v addopts = -s -v —html = report.html —return 3 # 终端运行命令个
testpaths = ./testCase # 需要执行的用例文件路径
python_files = test.py # 需要执行的文件格式
python_classes = Test # 需要执行的类的格式
python_functions = test* # 需要执行的方法的格式
